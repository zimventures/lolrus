## Capturing Screenshots

1. Run lolrus with test credentials
2. Use a screen capture tool (Windows: Snipping Tool, macOS: Cmd+Shift+4, Linux: gnome-screenshot)
3. For animated GIFs, use tools like:
   - Windows: ScreenToGif
   - macOS: GIPHY Capture, Kap
   - Linux: Peek, gifski
4. Save files with the naming convention above
5. Optimize images to keep file sizes reasonable (<500KB for PNGs, <2MB for GIFs)
